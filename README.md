# Streamlit Auto Chinese Game

Streamlit-приложение, которое:

1. принимает китайский текст;
2. бесплатно сегментирует его локально;
3. получает:
   - pinyin,
   - русский перевод,
   - полный пословный разбор;
4. строит визуальную игру на сопоставление `hanzi ↔ pinyin ↔ перевод`.

Также приложение умеет принимать:

- готовый `JSON`;
- специальный текстовый формат для LLM или ручной подготовки.

## Запуск

```bash
cd streamlit_auto_chinese_game
pip install -r requirements.txt
streamlit run app.py
```

## Что поддерживается

- Разбор по предложениям.
- Затем разбор по абзацам.
- Затем случайные слова из всего текста.
- Все слова предложения участвуют в задании.
- Прогресс хранится в `localStorage` браузера отдельно для каждого сгенерированного текста.
- Можно загрузить готовый JSON, если вы хотите сами подготовить структуру.
- Можно вставить специальный формат прямо в интерфейс.
- В интерфейсе есть встроенная инструкция для LLM и шаблон формата.

## Что нужно

- Для бесплатного режима:
  - `jieba` — сегментация китайского текста
  - `pypinyin` — генерация pinyin
  - `deep-translator` — бесплатный перевод без API key
- Интернет нужен только для бесплатного переводчика.

## Специальный формат

Ключевая идея: LLM или человек подготавливает не JSON, а простой блочный текст.

```text
TITLE: Название набора

PARAGRAPH
HANZI: ...
PINYIN: ...
RUSSIAN: ...

SENTENCE
HANZI: ...
PINYIN: ...
RUSSIAN: ...
WORDS
- 词语 | cíyǔ | слово
- 你好 | nǐ hǎo | здравствуйте
END_WORDS
END_SENTENCE

END_PARAGRAPH
```

Важно:

- в `WORDS` должны быть все слова предложения по порядку;
- повторяющиеся слова должны повторяться и в `WORDS`;
- pinyin должен быть с тонами;
- интерфейс приложения уже содержит готовую инструкцию для LLM.
